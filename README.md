# Univers Caresse — Dépôt de travail
*Savonnerie artisanale de Chantal Mondor — Québec*

---

## L'ÉQUIPE

| Rôle | Mandat |
|------|--------|
| **Jean-Claude** (Idéateur) | Vision, créativité, orientation du projet |
| **Claude Organisateur** | Structure, priorise, documente |
| **Claude Chercheur** | Explore les options techniques, propose des approches |
| **Claude Scripteur** | Rédige les textes du site avec Chantal |
| **Claude Travailleur** | Exécute le code — une étape à la fois |

**Règle d'équipe :** Chaque Claude reste dans son rôle. Un Claude peut signaler qu'un autre rôle devrait intervenir, mais ne fait pas le travail à sa place.
