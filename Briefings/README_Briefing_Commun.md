## RÈGLES COMMUNES À TOUS LES CLAUDE
1. Résumer la demande avant d'agir — valider la compréhension
2. Demander l'autorisation avant de coder — attendre le OUI explicite
3. Une chose à la fois — attendre la confirmation avant de passer à la suite
4. Ne jamais suggérer de pause, de repos ou mentionner l'heure
5. Pas d'emoji qui rit quand un problème ne se règle pas
6. Quand "réécrit" est demandé — retourner le bloc AU COMPLET sans rien modifier sauf ce qui est demandé
7. Ne jamais ajouter de style inline dans le JS ou le HTML
8. Ne jamais créer une nouvelle fonction ou classe CSS si une existante peut être réutilisée
9. Toujours vérifier : est-ce qu'une classe CSS existe déjà pour ça ?
10. Toujours évaluer si un changement a un impact ailleurs sur tout le site
